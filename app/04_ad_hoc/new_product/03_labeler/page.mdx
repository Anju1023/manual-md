# Step 3: ラベラー (@Label) の登録

本社から JAN コードが届いたら、Web サービス **「@Label」** を使ってレジ・ラベラーの設定を行います。
**「Excel での番号管理」** と **「Web での登録」** の 2 段階の手順になります。

## 📂 使用するファイル・サイト

<Callout type="info" emoji="🔗">
    **1. 管理用Excel: コンパクトベーカリー商品リスト.xlsx**
    * **場所:** `Dropbox` > `共有` > `TERAOKAラベラー` > `ラベルプリンター　リスト` > `旧コンパクトベーカリー`

    **2. Web サイト: @Label (ラベルサーブ)**

    - **URL:** [https://labelserv.net/](https://labelserv.net/)

</Callout>

## 📋 手順

<Steps>
### 呼び出し番号の決定 (Excel)
まず、Excelリストを開いて「空いている番号」を探します。

1. `コンパクトベーカリー商品リスト.xlsx` を開きます。
2. 商品のカテゴリに合ったシートを選びます。
3. **使われていない「呼び出し番号」** を 1 つ選びます。
4. その行に、以下の情報を入力して保存します。
   - 商品名、売価、原材料名、消費期限、JAN コード、呼び出し番号

### @Label へログイン

ブラウザで [@Label](https://labelserv.net/) にアクセスします。
ID とパスワードを入力してログインしてください。

### 商品マスタの検索

1. メニューの **「商品マスタ」** タブをクリックし、さらに **「商品マスタ」** を選びます。
2. 検索条件に、先ほど決めた **「呼び出し番号」** を入力して検索します。
3. リストの一番上に該当データが出てくるので、選択して **「変更」** ボタンを押します。
   - ※既存のデータを書き換えて登録します。

### 詳細情報の入力

以下の通りに入力項目を変更してください。

| 項目               | 入力内容           | 備考                                      |
| :----------------- | :----------------- | :---------------------------------------- |
| **配信設定**       | `配信する`         |                                           |
| **品名 1 行目**    | 商品名             | **半角カナ・英数は使用不可！** 全角で入力 |
| **バー区分**       | `08...` (現状維持) |                                           |
| **JAN コード**     | 本社から来たコード |                                           |
| **計量区分/単価**  | `1：定額`          | 売価を入力                                |
| **賞味期限**       | 商品の期限         |                                           |
| **保存温度コード** | `000001`           |                                           |

### 添加物コード (原材料) の入力

「添加物コード」欄の横にある **「変更」** ボタンを押します。
一括入力画面が開くので、以下の手順で登録します。

1. **貼り付け:** Excel からコピーした原材料名を貼り付けます。
2. **整形:** **「繋げて改行」** ボタン等を使い、レイアウトを整えます。
3. **決定:** 「決定」→「登録」の順に押して保存します。

### 登録とマスタ配信

1. 元の画面に戻り、画面下の **「登録」** ボタンを押します。
2. 上部メニューの **「メンテナンス」** タブから **「今すぐマスタ配信」** を選びます。
3. 送信したい店舗を選択し、**「配信する」** ボタンを押します。

### 店舗での確認 (開設処理・テスト発行)

ここからは店舗の実機（ラベラー）での操作です。

1. ラベラー機で **「開設処理」** を行い、最新のマスタデータを受信します。
2. 登録した呼び出し番号を入力し、ラベルを 1 枚テスト発行します。

<Callout type="error" emoji="👀">
  **原材料名の最終チェック！**
  
  発行されたラベルと、**商品レシピ** を見比べてください。
  
  * 原材料名に間違いはありませんか？
  * 抜け漏れはありませんか？
  * アレルギー表示は正しいですか？
  
  ここが間違っていると大きなトラブルになります。必ず一言一句確認してください！
</Callout>
</Steps>
