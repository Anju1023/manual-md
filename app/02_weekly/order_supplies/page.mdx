# 📦 消耗品の発注（火曜・木曜）

週に 2 回、エフピコさんへ消耗品（トレーや袋など）の発注を行います。

## 🕒 作業タイミング

- **曜日:** 毎週 火曜日・木曜日
- **担当:** 早番の人が実施

<Callout type="warning" emoji="⚠️">
	**重要:** 必ず午前 10:00 までにメール送信を完了させてください！
</Callout>

## 📋 手順

### 1. ファイルを開く

デスクトップにあるショートカット `エフピコ発注書.xlsx` を開きます。

### 2. 日付の計算式を貼り付け

以下の数式をコピーして、Excel の **J4 セル** をコピーし、**A2 セル** に「値貼り付け」してください。

```excel
=LET(
    今日, TODAY(),
    曜日, WEEKDAY(今日),
    納品日, IFS(曜日=3, 今日+2, 曜日=5, 今日+3, TRUE, "エラー"),
    TEXT(納品日, "yyyy/mm/dd")
)
```

### 3. 発注数の入力

在庫を確認して、必要な数量を入力します。

### 4. メール送信

江藤さん・日下部さん宛にファイルを添付して送信します。
