# 🎛️ 操作シート・ボタン機能一覧

「操作シート」に配置されている全 14 個のボタンの機能解説です。
各ボタンは VBA マクロを実行し、必要な帳票の出力や集計を行います。

## 🚀 一括出力ボタン (推奨)

基本的には、これらのボタンを使ってまとめて出力します。

| ボタン名                       | 実行マクロ                   | 機能・用途                                               |
| :----------------------------- | :--------------------------- | :------------------------------------------------------- |
| **合計まとめて出力**           | `合計まとめて出力`           | 製造数合計などを一括で処理します。                       |
| **まとめてラベル出し出力**     | `まとめてラベル出し出力`     | 消費期限チェック表などを一括出力します。                 |
| **生地出し並べ表まとめて出力** | `生地出し並べ表まとめて出力` | 「冷凍生地」「スクラッチ」「並べる表」を一括出力します。 |

## 🏭 個別出力ボタン (製造・管理用)

特定の帳票だけを出したい場合に使用します。

### 製造数・集計

| ボタン名                 | 実行マクロ                 | 内容                               |
| :----------------------- | :------------------------- | :--------------------------------- |
| **CB 製造数出力**        | `CB製造数合計出力`         | センターベーカリー全体の製造数合計 |
| **クリエイト製造数出力** | `クリエイト製造数合計出力` | クリエイト納品分の製造数のみ抽出   |
| **製造数出力**           | `製造数合計出力`           | (総合計) 製造指示数                |

### 工程表 (生地・成形)

| ボタン名                     | 実行マクロ                 | 内容                               |
| :--------------------------- | :------------------------- | :--------------------------------- |
| **冷凍生地出し表出力**       | `冷凍生地出し表出力`       | 前日出しの冷凍生地リスト           |
| **スクラッチ生地出し表出力** | `スクラッチ生地出し表出力` | 当日仕込み（スクラッチ）生地リスト |
| **並べる表出力**             | `並べる表出力`             | 天板への並べ個数・枚数リスト       |

### ラベル・期限管理

| ボタン名                       | 実行マクロ                         | 内容                               |
| :----------------------------- | :--------------------------------- | :--------------------------------- |
| **ラベル出し表出力**           | `消費期限チェック表出力`           | ローゼン用ラベル・期限チェック表   |
| **クリエイトラベル出し表出力** | `クリエイト消費期限チェック表出力` | クリエイト用ラベル・期限チェック表 |

### 納品・発注

| ボタン名                 | 実行マクロ                             | 内容                         |
| :----------------------- | :------------------------------------- | :--------------------------- |
| **ローゼン商品別出力**   | `商品別シート出力`                     | 店舗別・商品別の仕分けリスト |
| **クリエイト商品別出力** | `クリエイト商品別シート出力`           | クリエイト用の仕分けリスト   |
| **発注表出力**           | `発注表作成`<br/>`発注表_仕入先別出力` | 原材料の発注書を作成・出力   |

---

## 🔧 開発者向けメモ

- **一括ボタンの仕組み:** `Call` ステートメントで、個別の出力マクロ (`Call 冷凍生地出し表出力` 等) を順番に呼び出しています。
- **修正時の注意:** 個別のマクロを修正した場合、「まとめて出力」の結果にも影響します。
