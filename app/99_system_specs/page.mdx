# システム仕様書 (管理者向け)

本ページは、システム管理者および開発者向けの技術資料です。
マクロの構造、ファイルパス、トラブルシューティング情報を記載しています。

## コンテンツ

- **ファイルパス一覧**
  - 業務で使用するファイルの絶対パスおよびショートカット配置
- **VBA マクロ解説**
  - 自動化処理のロジック説明
  - デバッグ手順
- **数式辞書 (Excel Formulas)**
  - 発注日計算ロジック (LET 関数など)
  - シフト計算ロジック
- **トラブルシューティング**
  - セキュリティ警告への対処
  - 既知のエラーと回避策
